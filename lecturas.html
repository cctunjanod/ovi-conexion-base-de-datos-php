CONEXION A BASES DE DATOS EN PHP

Es una gran herramienta ya que por medio de esta se puede hacer la página más dinamica además de permitir guardar datos, y tener control sobre estos según como sea necesario, cabe destacar que para el uso de PHP se necesita de un servidor para que este funcione; Uno de los más usado es LOCALHOST el cual se ejecuta mediante XAMPP o WAMP, esto con el fin de ejecutar el codigo PHP y tener la certeza de que trabaja sin problema alguno.

SENTENCIA SELECT

Nos permite hacer consultas en una o más tablas de una base de datos, esto con el fin de mostrar o extraer la información que se necesite.

X

SENTENCIA INSERT

Con esta sentencia se ingresan nuevas filas en la tabla que se establezca en la consulta, de esto modo se "alimenta" la base de datos, teniendo información almacenada en ella.

X

SENTENCIA UPDATE

Mediante esta sentencia se puede efectuar cambios en los datos que ya están en la tabla, por lo cual si durante el ingreso de los datos hubo un error pues de esta forma se puede reparar, o cual es una gran ventaja a la hora de tener los datos correctos en la base de datos.

X

SENTENCIA DELETE

Con esta sentencia se eliminan todos los datos de la tabla que se especifique, ya que al momento de especificar se toma de referencia toda la fila y por ende sus datos.
