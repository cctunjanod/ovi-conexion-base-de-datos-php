  • Información General

    • Para la realización de este trabajo se procedió a crear una página web con el objetivo de dar a conocer como se desarrolla la implementación de una conexión de base de datos mediante PHP, además de esto también se mencionan las herramientas CRUD que son de gran importancia en el ámbito de las conexiones con bases de datos.

      Por esto, la página web consta de varias herramientas divididas por secciones en las que se explicará de manera sencilla y detallada cada una de las funciones anteriormente mencionadas, con la ayuda de las diferentes herramientas que tiene la página, en las cuales hay contenido multimedia y un apartado donde se podrá poner a prueba el conocimiento que se tiene con relación al tema.

  • Objetivos

    • Enseñar como realizar la conexion de una BD a través de PHP

    • Evidenciar las pricipales sentencias que se usan en relación con las bases de datos

    • Ofrecer diferentes medios para el aprendizaje

    • Poner a prueba los conocimientos adquiridos mediante las actividades propuestas

  • Tematicas

    • La tematica de la página se centra en como establecer la conexion con una base de datos mediante PHP, sin embargo también se abarcan las sentencias que se encuentran embebidas en el tema central por lo que es de gran importancia hablar de ellas y sus funciones y como ejecutarlas.
